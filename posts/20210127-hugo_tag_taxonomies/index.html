Hugo лё”лЎңк·ё нҸ¬мҠӨнҠё л¶„лҘҳлІ•(taxonomies)

TaxonomyлһҖ

л¶„лҘҳлІ•. л°°м—ҙ(arrangement)лҘј лң»н•ҳлҠ” taxisм—җ лІ•(law)мқ„ лң»н•ҳлҠ” nomosк°Җ кІ°н•©н•ң лӢЁм–ҙлӢӨ. мғқл¬јн•ҷм—җм„ң мў…мҶҚкіјлӘ©к°•л¬ёкі„, мҳҒнҷ”м—җм„ң мһҘлҘҙ, лё”лЎңк·ём—җм„ң нғңк·ёмҷҖ м№ҙн…Ңкі лҰ¬, мқҙ лӘЁл‘җ л¶„лҘҳлІ•мқҳ мў…лҘҳлӢӨ.

л¶„лҘҳлҠ” кі§ кө¬мЎ°лӢӨ. лӮңмһЎн•ҳкІҢ нқ©м–ҙм ё мһҲлҠ” мҪҳн…җмё л“Өм—җ мҲңм„ңлҘј л§Ңл“Өкі  м •л ¬н•ңлӢӨ. кө¬мЎ°лҘј мң„н•ң кё°мӨҖмқ„ м •н•ҳлҠ” мқјмқҖ мүҪм§Җ м•ҠлӢӨ. лё”лЎңк·ёлҘј мҡҙмҳҒн•ҙліё мӮ¬лһҢмқҙлқјл©ҙ м№ҙн…Ңкі лҰ¬лҘј лӘҮ лІҲмқҙлӮҳ л°”кҫёлҠ” кІҪн—ҳмқ„ н–Ҳмқ„ кІғмқҙлӢӨ. лҳҗлҠ” нғңк·ёлҘј л¬ҙн•ңм • м°Қм–ҙлӮҙлӢӨк°Җ м№ҙмҳӨмҠӨлҘј л§ҲмЈјн•ҳкё°лҸ„ н•ңлӢӨ.

м Ғм Ҳн•ң кө¬мЎ°лҘј л§Ңл“Өкё° мң„н•ҙ мҳ¬л°”лҘё м „лһөмқ„ к°–кі  л¶„лҘҳлІ•мқ„ м •н•ҙм•јн•ңлӢӨ. HugoлҠ” м–ҙл–»кІҢ TaxonomyлҘј лӢӨлЈЁлҠ”м§ҖлҘј нҷ•мқён•ҳкі  л¶„лҘҳлІ•м—җ лҢҖн•ҙ лӢӨлӨ„ліҙл Ө н•ңлӢӨ.

Hugoмқҳ мҪҳн…җмё  л¶„лҘҳлІ•(Taxonomy)

HugoлҠ” мӮ¬мҡ©мһҗк°Җ м •мқҳм—җ л”°лқј мҪҳн…җмё лҘј л¶„лҘҳн•ҳлҠ” кё°лҠҘмқ„ м ңкіөн•ңлӢӨ.

Taxonomy

л¶„лҘҳмқҳ кё°мӨҖмқ„ Taxonomyлқјкі  н•ңлӢӨ. мҳҒнҷ” лё”лЎңк·ёлҘј мҳҲмӢңлЎң л“Өл©ҙ л°°мҡ°, к°җлҸ…, м ңмһ‘мӮ¬, мһҘлҘҙ л“ұмқҙ Taxonomyк°Җ лҗңлӢӨ. н•ҳлӮҳмқҳ TaxonomyлҠ” Termкіј ValueлЎң мқҙлЈЁм–ҙм ё мһҲлӢӨ.

Term

TermмқҖ Taxonomyм—җ мҶҚн•ҳлҠ” нӮӨмӣҢл“ңлӢӨ. мҳҲлҘј л“Өм–ҙ мһҘлҘҙк°Җ Taxonomyлқјл©ҙ лЎңл§ЁмҠӨ, нҢҗнғҖм§Җ, нҳёлҹ¬, м•Ўм…ҳ л“ұмқҙ Termмқҙ лҗңлӢӨ.

Value

ValueлҠ” Termм—җ л”°лҘё кІ°кіјк°’мқ„ л§җн•ңлӢӨ. мһҘлҘҙ:лЎңл§ЁмҠӨлҘј нҒҙлҰӯн•ҳл©ҙ л№„нҸ¬м„ лқјмқҙмҰҲ, мқҙн„°л„җ м„ мғӨмқё л“ұмқҳ мҳҒнҷ” лҰ¬мҠӨнҠёлҘј л°ҳнҷҳн•ңлӢӨ.

 мһҘлҘҙ <- Taxonomy
 	лЎңл§ЁмҠӨ <- Term
 		л№„нҸ¬ м„ лқјмқҙмҰҲ <- Value
 		мқҙн„°л„җ м„ мғӨмқё <- Value
 	нҢҗнғҖм§Җ <- Term
 		н•ҙлҰ¬нҸ¬н„°мҷҖ м•„мҰҲм№ҙл°ҳмқҳ мЈ„мҲҳ <- Value
 		л°ҳм§Җмқҳ м ңмҷ•: л‘җк°ңмқҳ нғ‘ <- Value

нҸ¬мҠӨнҠёмқҳ нғңк·ёмҷҖ м№ҙн…Ңкі лҰ¬ м§Җм •н•ҳлҠ” л°©лІ•

Hugoмқҳ кё°ліё TaxonomyлҠ” нғңк·ё(tag)мҷҖ м№ҙн…Ңкі лҰ¬(category)лӢӨ. Config нҢҢмқјм—җ нғңк·ёмҷҖ м№ҙн…Ңкі лҰ¬лҘј taxonomyлЎң м§Җм •н•ҳм§Җ м•Ҡм•„лҸ„ HugoлҠ” мқҙлҘј кё°л°ҳмңјлЎң мҪҳн…җмё лҘј л¶„лҘҳн•ңлӢӨ. нҸ¬мҠӨнҠёмқҳ л©”нғҖлҚ°мқҙн„°лҘј лӢҙмқҖ frontmatterм—җ categoryмҷҖ tagлҘј м Ғм–ҙмЈјл©ҙ лҒқмқҙлӢӨ.
